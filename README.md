# Vicenzino Family Christmas Site

A festive Christmas website for the Vicenzino family, featuring a countdown, photo gallery, and advent calendar.

## Getting Started

This project is built with React and Vite.

### Prerequisites

- Node.js installed on your machine.

### Installation

1.  Open your terminal in this directory.
2.  Run `npm install` to install dependencies.
3.  Run `npm run dev` to start the local development server.

### Features

-   **Snowfall Animation**: A magical snowy background.
-   **Christmas Card**: A 3D interactive card.
-   **Countdown**: Counts down the days, hours, minutes, and seconds to Christmas.
-   **Gallery**: Upload and display family photos (Polaroid style).
-   **Advent Calendar**: A 12-day interactive calendar.

## Project Structure

-   `src/App.jsx`: Main application logic and components.
-   `src/index.css`: Styles and animations.
